## プロジェクト概要

「現場適応されているデジタル施策をアップデートする検討」および「現場最適化された、作業手順書を作成するシステムの構築」を目的としたプロジェクト・ワークスペースです。

## 現在のフェーズ: 構想・検討フェーズ

現在はシステムの実装は行わず、以下を主な活動としています。

- 現状の課題整理
- 関連システム(`High-precision-sensor`, `solar-aldrin`)の仕様・構成調査
- 新システムの要件定義および設計思想の検討
